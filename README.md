# Retail Sales Optimization — ABC Retail Corp

## Overview
End-to-end data engineering capstone project using the Rossmann Store Sales
dataset. Built a full ETL pipeline, star schema data warehouse, and a Random
Forest sales forecasting model achieving 13.16% RMSPE.

## Project structure
retail_project/
├── data/raw/              ← original Rossmann CSV files
├── data/processed/        ← generated by pipeline (gitignored)
├── pipeline/              ← ETL scripts (extract, transform, load)
├── notebooks/             ← Jupyter notebooks (01–05)
├── reports/               ← charts, reports, presentation
└── logs/                  ← pipeline run logs

## How to run
1. Download Rossmann dataset from Kaggle to data/raw/
2. pip install pandas numpy scikit-learn matplotlib seaborn
3. python pipeline/run_pipeline.py
4. Open notebooks/ in Jupyter and run in order (01 → 05)
